Transform

A propriedade transform aplica sobre transformações 2D e 3D em um elemento. Esta propriedade permite mover, rotacionar, escalar, inclicar, entre outros. Abaixo segue os valores possíveis para esta propriedade.

matrix(n, n, n, n, n, n)

Aplica uma tranformação 2D, utilizando uma matriz de 6 valores.

Exemplo

matrix3d(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)

Aplica uma tranformação 3D, utilizando uma matriz 4x4 de 16 valores.

translate(x,y)

Define uma translação 2D.

Exemplo

translate3d(x,y,z)

Define uma translação 3D.

translateX(x)

Faz uma translação 2D apenas no eixo x.

Exemplo

translateY(y)

Faz uma translação 2D apenas no eixo y.

Exemplo

translateZ(z)

Faz uma translação 3D apenas no eixo z.

scale(x,y)

Define uma tranformação de escala 2D.

Exemplo

scale3d(x,y,z)

Define uma tranformação de escala 3D.

scaleX(x)

Faz uma tranformação de escala 2D apenas no eixo x.

Exemplo

scaleY(y)

Faz uma tranformação de escala 2D apenas no eixo y.

Exemplo

scaleZ(z)

Faz uma tranformação de escala 3D apenas no eixo z.

rotate(angle)

Define uma rotação 2D, utilizando o ângulo (em deg) passado como parâmetro.

Exemplo

rotate3d(x,y,z,angle)

Define uma rotação 3D.

rotateX(angle)

Define uma rotação 3D no eixo x.

Exemplo

rotateY(angle)

Define uma rotação 3D no eixo y.

Exemplo

rotateZ(angle)

Define uma rotação 3D no eixo z.

Exemplo

skew(x-angle,y-angle)

Define uma transformação de inclinação 2D nos x e y.

Exemplo

skewX(angle)

Define uma transformação de inclinação 2D no eixo x.

Exemplo

skewY(angle)

Define uma transformação de inclinação 2D no eixo y.

Exemplo

perspective(n)

Define uma visão perspectiva para um elemento 3D transformado.

initial

Seta a propriedade para o seu valor padrão.

inherit

Herda esta propriedade do seu parent.
